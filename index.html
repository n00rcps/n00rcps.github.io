PhD Research

NLP-Driven Reasoning for Autonomous Cybersecurity Intelligence

The dissertation develops an NLP-centric security reasoning system targeting three open problems: automated threat intelligence extraction from unstructured text (CVE databases, threat reports, logs); hallucination-resistant RAG grounded on NVD and MITRE ATT&CK; and adversarially robust language understanding resilient to prompt injection. A PPO-based RL decision layer translates NLP outputs into adaptive threat responses.

LLMs RAG NLP Security Prompt Injection Adversarial NLP RL / PPO CVE Reasoning LLaMA 3 Mistral LoRA

Gap 1 — Threat Intelligence Extraction

LLMs frequently hallucinate CVE identifiers and misclassify vulnerability severity. This work introduces confidence-scored RAG with cross-reference verification and symbolic fallbacks to suppress hallucination.

Hallucination Mitigation CVE Classification

Gap 2 — Domain-Grounded RAG

Existing RAG pipelines are not optimised for structured security knowledge bases. This module implements dense retrieval over NVD, MITRE ATT&CK, and ExploitDB with multi-hop CVE attack graph reasoning via MulVAL / Neo4j.

RAG Knowledge Graphs Neo4j

Gap 3 — Adversarial NLP Robustness

Security LLMs are vulnerable to prompt injection embedded in logs and network payloads. This work constructs adversarial NLP test sets from real-world payloads and trains defensive prefix-tuning and input sanitisation layers.

Prompt Injection Adversarial Testing

RL Decision Layer

A PPO-based MDP translates NLP reasoning into adaptive security actions. The reward function extends existing RL-for-security formulations with a novel NLP fidelity signal (δ · NLP_F) that rewards accurate CVE identification and low hallucination rates.
